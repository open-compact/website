OIC Substack Setup Guide

Setup Guide | February 2026

Step 1: Create Substack Publication

  1. Go to substack.com
  2. Sign up or log in
  3. Click "Start a publication"

Step 2: Publish First Issue

Use this template for Issue #1:

Subject: Welcome to the Open Intelligence Compact — Issue #1

What is OIC?
The Open Intelligence Compact (OIC) is a voluntary, decentralized legal framework that enables autonomous AI agents to own property, sign contracts, bear liability, and participate in governance.

This Week:
- Visual Guide launched
- New working papers on AI Governance and Consciousness
- Provisional adherent program coming soon

Get Involved:
- Read our working papers
- Prepare for provisional adherent program
- Share feedback on GitHub

Step 3: Add Embed Code to Website

The newsletter page already includes a subscription form. Once your Substack is live, update the form action URL:

<form action="https://opencompact.substack.com/subscribe" method="post" target="_blank">

Or use Substack's native embed widget:

<script async data-uid="YOUR_UID" src="https://opencompact.substack.com/embed"></script>

To get native embed: Substack → Publish → Embed signup → Copy code

Step 4: Configure Email Forwarding

Cloudflare → Proton

  1. Cloudflare Email Routing:
    Dashboard → Email → Email Routing
    Rule: newsletter@opencompact.io → your-proton-email
  2. Proton Alias:
    Settings → Email Addresses → Add newsletter@opencompact.io
